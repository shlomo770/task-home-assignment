# AI Usage / Prompt Notes

During the development of this assignment, ChatGPT was used as a technical assistance tool for research, clarification, brainstorming and improving implementation quality.

AI assistance was mainly used for:

 Redux Toolkit 
 Async flows (`Promise`, `async/await`)
 SSE / WebSocket lifecycle handling
 Redux normalized state (`byId` / `allIds`)
 DTO → Domain mapping
 Retry / reconnect strategies
 TypeScript typing improvements
 Separation of concerns between application layers

In addition, since the assignment referenced a mock server that was not provided, AI assistance was also used to help create a temporary local mock server based on the assignment requirements in order to continue development and integration work.

AI was used strictly as a professional assistance tool.

The system architecture, integrations, debugging process and implementation decisions were reviewed and implemented manually throughout the development process.
All integrated code was reviewed, adapted and understood as part of the assignment implementation.
